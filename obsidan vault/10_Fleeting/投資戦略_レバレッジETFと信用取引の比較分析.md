# 投資戦略_レバレッジETFと信用取引の比較分析

## アイデア

レバレッジETFと信用取引のどちらを選ぶべきか、手数料・リスク・効率の観点から比較分析した手書きメモの整理。

### 用語定義

| 略語 | 正式名称 | 値（例） |
|------|----------|----------|
| **SLR** | Spot Leverage Ratio（現物レバレッジ比率） | 2 |
| **MLR** | Margin Leverage Ratio（信用レバレッジ比率） | 1 |
| **MRR** | Margin Ratio Requirement（証拠金維持率） | 0.3 (30%) |
| **iMRR** | Initial MRR（新規建て時の必要証拠金率） | 0.3 (30%) |
| **MR** | Margin Ratios（マージン比率） | 0.3 |
| **ARC** | Assessment Rate of Collateral（担保評価率） | 0.8 (80%) |

### 保証維持率の計算式

$$保証維持率 = \frac{代金 + 評価差}{建玉}$$

### 戦略の選択基準

> [!tip] シンプルな判断基準
> - **現物** → レバレッジETF一択
> - **信用** → レバレッジを選べる場合はレバレッジ、無理なら1倍

### 2つのケース比較

| 項目 | 現レバ + 信1倍 | 現1倍 + 信レバ |
|------|----------------|----------------|
| **初期保証維持率** | 30% | 60% |
| **保維インパクト** | 1.6x | 3.2x |
| **損益インパクト** | 4.67x | 4.67x |
| **追証乖離** | -6.25% | -12.5% |
| **ETF手数料（1倍）** | 0.0561% | - |
| **ETF手数料（2倍）** | 0.575% | - |
| **ETF手数料合計** | 0.975% | 1.925% |
| **信用金利** | 7.486% | 3.73% |
| **合計手数料** | 8.44% | 5.65% |
| **1取引倍率** | 2.6倍 | 1.3倍 |

> [!warning] 注意点
> - 信用取引は金利コストが大きい
> - レバレッジETFは経費率（信託報酬）が発生
> - 追証乖離が大きいほど強制決済リスクが高い

### 結論・考察

1. **合計手数料**は「現1倍 + 信レバ」の方が低い（5.65% vs 8.44%）
2. **追証リスク**は「現レバ + 信1倍」の方が低い（-6.25% vs -12.5%）
3. **効率性**を重視するなら「現1倍 + 信レバ」
4. **安全性**を重視するなら「現レバ + 信1倍」
