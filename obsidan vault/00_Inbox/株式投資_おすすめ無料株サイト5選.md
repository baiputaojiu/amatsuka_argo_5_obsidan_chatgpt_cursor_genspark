# おすすめ無料株サイト5選｜情報収集から需給分析まで

> [!summary] 概要
> 株式投資に役立つ無料サイト・ツールを5つ紹介。マクロ（全体感）からミクロ（個別株）まで、効率的に情報収集する方法を解説。

---

## 1. X（旧Twitter）

> [!tip] 最強の情報収集ツール
> 株以外の情報も含め、幅広く収集できる

### メリット
- リアルタイムで情報が入手できる
- 業界の中の人の声が聞ける
- 個人投資家の雰囲気が分かる
- トレンド機能で話題の銘柄が分かる

### デメリット・注意点

> [!warning] 鵜呑みにしない
> - 有益な情報から詐欺まで玉石混交
> - 「儲かってます」アピールは要注意
> - ボットによる誘導リンクに注意
> - 時価総額100億以下の煽りは仕手株の可能性

### おすすめの使い方

#### 1. 気になる銘柄を検索
```
検索例：「銘柄名 + 株」
```
→ その銘柄に対する株式投資的な考察が見つかる

#### 2. 業界の中の人のポストを見る

> [!example] 具体例
> - ポケポケ（[[DeNA]]）が流行っている
> - Xで「面白い」「課金してる」という声が多い → 株価上昇期待
> - 逆に「つまらない」が多いのに株価上昇中 → 下落リスク

#### 3. 個人投資家の雰囲気を把握

| 状況 | 意味 |
|------|------|
| みんな儲かってる投稿が多い | **天井が近い** |
| 悲観的なコメントが多い | **底打ちが近い** |
| 投稿が減っている | **注目度低下** |

#### 4. 決算またぎ前の確認

> [!danger] 決算ギャンブル前の必須チェック
> - 「自分だけが知ってる銘柄」と思っていても、意外とみんな注目している
> - 注目度が高い銘柄は「好決算 → 出尽くし売り」になりやすい
> - 特に**時価総額500億前後 + 中小型 + カタカナ社名**は要注意

## 2. ブルームバーグ（Bloomberg）

> [!info] 金融関係者必読のニュースサイト
> https://www.bloomberg.co.jp/

### 特徴
- 世界中の金融関係者が見ている
- 記事が出ると株価が動くことも
- グローバルな相場感が掴める

### 見るべきポイント
- 株価が上下した理由の解説
- ブルームバーグなりの見解
- アメリカ・世界の情報

> [!quote] 使い方
> 「金融関係者のプロはこういう風に今のマーケットを見てるんだな」という視点で読む

---

## 3. カブタン（株探）

> [!tip] 個別株分析の総合ツール
> https://kabutan.jp/

### 見れる情報
- 通期・四半期の業績
- 財務状況・収益性
- 信用残データ
- 株主構成
- 修正履歴

### おすすめの使い方

#### 1. 業績でスクリーニング
```
売上20%成長 + 利益30%成長 → 気になる銘柄を発見
→ 決算短信・説明資料を深掘り
```

#### 2. 値上がり・値下がりランキング
- 「なぜこの銘柄が動いているのか？」を調べるきっかけに

#### 3. 決算速報

> [!note] 決算シーズンの必須ツール
> - 上方修正・増配などがリアルタイムで分かる
> - 決算シーズンはアクセス集中でサイトが重くなることも

#### 4. PTSランキング
- 夜間取引の上昇銘柄 → 翌日も上がる可能性
- 10%以上上昇 → 何らかの材料あり

#### 5. 修正履歴（プレミアム会員）
- その会社の「癖」が分かる
- 毎年同時期に上方修正する会社など

## 4. 世界の株価

> [!info] 総合相場感ツール
> https://sekai-kabuka.com/

### 見れる情報

| カテゴリ | 内容 |
|---------|------|
| 株価指標 | 日経平均PER、寄与度など |
| 業種別 | 業種別騰落率 |
| 需給 | 空売り比率、投資主体別動向 |
| 商品 | VIX、原油、金、銀、小麦など |
| その他 | 債券価格、仮想通貨、日経先物 |

### 特に注目：6日騰落レシオ

> [!tip] 全体の相場感を測る指標
> - **100が基準**
> - 高い（120以上）= 上がりすぎ → 利確タイミング
> - 低い（60以下）= 下がりすぎ → 逆張りチャンス

```
上げすぎ ──┐     ┌── 上げすぎ
           ↓     ↓
    ───────────────── 100
           ↑     ↑
下げすぎ ──┘     └── 下げすぎ
```

## 5. 松井証券

> [!info] 需給確認 + IPO申込に便利
> https://www.matsui.co.jp/

### 特徴1: IPO抽選時に入金不要

> [!tip] 大きなメリット
> - 通常の証券会社：申込時に資金ロックが必要
> - 松井証券：**当選してから入金でOK**
> - とりあえず申し込んでおくことが可能

### 特徴2: 株アプリの売買分析機能

#### 現物 vs 信用取引の比率が分かる

> [!example] 銘柄別の特徴
> | 銘柄 | 現物比率 | 特徴 |
> |------|---------|------|
> | [[三菱商事]] | 90% | ガチホ勢が多い、動きが緩やか |
> | [[レーザーテック]] | 50% | 短期資金が多い、上下が激しい |

#### 空売り比率が分かる

| 比率 | 解釈 |
|------|------|
| 20%前後 | 通常 |
| 30%以上 | やや多め |
| 50%以上 | 異常に多い（機関の空売り） |

> [!warning] 空売り比率が高い銘柄の例
> - [[オリエンタルランド]]: 48%
> - [[資生堂]]: 53%
> - → ヘッジファンドが大量に空売り中
> - → ほとぼりが冷めるまで触りにくい

## まとめ：ツールの使い分け

```
【マクロ（全体感）】
  ブルームバーグ → 世界の相場感
  世界の株価 → 日本の相場感、業種別動向
      ↓
【ミクロ（個別株）】
  カブタン → 業績・財務分析
  X → 業界の声、投資家の雰囲気
  松井証券 → 需給分析
```
