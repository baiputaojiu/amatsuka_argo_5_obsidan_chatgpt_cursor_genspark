# 2026年以降の投資戦略：金・海外株・国内株

> 本ノートは、20_Literature の5本のレポート（2026年1月〜2月収録・執筆）と必要に応じたWeb情報を統合し、**金（貴金属）・海外株・国内株**の3軸で「今後有望と思しき投資戦略」を構築した永続ノートである。個別の買い売りや銘柄推奨ではなく、**テーマ・セクター・リスク・タイミング**の枠組みを整理する。

---

## 前提・共通認識（ソース間で一致している点）

- **インフレは継続**し、デフレに戻る可能性は極めて低い（通貨価値の減価・財政拡張・供給制約）。
- **高市政権の経済政策**（財政拡張・消費対策・骨太の方針の危機管理投資）は、少なくとも数年間は「高圧経済」を維持する方向で、**名目GDP・株価と連動**しうる。
- **衆議院選挙（2月8日）**の結果により週明け〜数ヶ月のシナリオが分かれるが、**与党過半数が確保される想定**で、2/3取得の有無で「為替介入の有無」「株価の振れ方」が変わる。
- **円安の構造要因**（貿易赤字・デジタル赤字・新規NISAの円売り）は為替介入では変わらず、**中長期の円安トレンド**は維持されうる。
- **金利**は日本・米国とも「下がらない」「上がりうる」との見方が強く、債券は目減りリスク、**PERが極端に高い成長株**は金利上昇時に脆弱。

## 1. 金（貴金属）の戦略

### 1.1 位置づけ

- **通貨価値の減価・地政学リスク・中央銀行のドル分散**に対する**実物ヘッジ**として、複数ソースで一貫して評価されている。
- ファンドマネージャー調査（BofA）では**ゴールドが最多保有（51%）**で、マグニフィセントセブンから資金がゴールドへシフトした構図が指摘されている。
- アナリスト調査（Reuters等）では2026年の金価格中央値が**約4,746ドル/トロイオンス**へ引き上げられており、中央銀行買い・インフレヘッジ・地政学が継続的な支えとされている。

### 1.2 戦略の要点

| 観点 | 内容 |
|------|------|
| **役割** | ポートフォリオの**保険・ヘッジ**。円建て資産の目減りや金融不安に対する代替資産。 |
| **保有形態** | 現物・ETFの両方が言及されている。銀は「ETFは売り・現物買い」の構図もあり、**ゴールドは通貨代替・インフレヘッジ**としてETFでも需要が持続。 |
| **銀** | ソーラーパネル需要（需要の20〜25%）・銅の副産物で供給非弾力性。**100ドル付近は心理的節目**で短期の調整リスクあり。短期的な投機より、**押し目での積立・長期保有**が無難。 |
| **銅・プラチナ・パラジウム** | データセンター・EV/ハイブリッド・水素など**産業需要**と地政学（南ア・ロシア供給）で言及。金・銀とあわせ「**コモディティ・貴金属**」として一定ウエイトを置く選択肢がある。 |

### 1.3 リスク・注意

- 短期で**過熱・急騰**しており、「上がりすぎ」を指摘する声あり。**一括購入より分散・押し目積立**が推奨される。
- 銀は**ボラティリティが高く**、節目（例：100ドル）前後では調整を想定し、ポジションサイズを小さくする運用が言及されている。

## 2. 海外株の戦略

### 2.1 マクロ・セクターの方向性

- **米国**：S&P500・ナスダックは年初来の調整局面が言及されている。**中小型・バリュー**へのローテーション、**素材・エネルギー・銀行**が相対的に強いとの指摘あり。
- **分散**：オールカントリーは米国比率が高く「分散にならない」との指摘。**S&Pとそれ以外を1対1程度**など、地域分散の見直しが推奨されている。
- **新興国**：ブラジル・メキシコ・南アフリカなどが「今年も強い」とされ、**分散・新興国**テーマは複数ソースで言及。

### 2.2 米国ハイテク・データセンター

- **スーパーマイクロ・NVIDIA・ブラックウェルウルトラ**に代表されるAIサーバー・データセンターは、**日米製造業の急回復**と結びつけてポジティブに語られている（ファンダメンタルズは改善）。
- 一方で**過剰投資リスク**が強調されている。ハイパースケーラー5社のうち、コンセンサスで**5年後も投資元本を回収できない可能性がある企業が2社前後**との指摘。**データセンターに投資している企業としていない企業（Appleは投資していない）**の分岐が重要。
- **2026年半ば以降**、金利上昇とあわせて米国ハイテクが「オールオッケー」ではなくなり、**銘柄選別・勝ち負けの明確化**が進むとの見方。日経平均は米国ハイテクのウェイトが大きいため、**中盤以降は警戒**する視点がある。

### 2.3 季節性・タイミング（鈴木のチャンネル）

- **2月**：ビットコイン・米国テックは軟調になりやすい。**米国税還付**が2月下旬にまとめて入る制度変更の年は、**2月24〜25日頃を「上げの得意日」**とする見方。それまでは神経質な局面。
- **2月下旬**：アジア休場時に**日本株が一人勝ち**しやすい。資金還付と重なると日本株がまず買われやすい。
- **6月〜7月頭**：米国は下げやすいが**日本株だけ上がる**季節性。自社株買い発表（4〜5月決算）と配当（5月下旬〜6月末）の再投資等が背景。

### 2.4 海外株のまとめ

- **長期**：米国・地域分散（S&P以外とのバランス）を維持しつつ、**バリュー・中小型・素材・エネルギー**にウエイトを置く選択肢がある。
- **ハイテク**：データセンター・AIは**銘柄選別**が重要。回収できる企業とできない企業の差がはっきりするという前提で、**過度な集中は避け、押し目で分散**する考え方が妥当。
- **タイミング**：2月は税還付前の不安定さ、6〜7月は日本株優位の季節性を意識した**再バランス**の参考にできる。

## 3. 国内株の戦略

### 3.1 選挙結果とシナリオ

- **与党2/3取得**：参議院が不要になる水準。**為替介入**が現実味を帯び、実施されれば**約10営業日で約10%の株安**の可能性が言及されている。その後の**中長期は株高**との整理。食品減税なども進みうる。
- **過半数だが2/3未満**：為替介入は出しにくく、**ストレートな株高**になりやすい。
- **過半数割れ**：複数ソースで**史上最大級の暴落リスク**が指摘されている。**選挙日前（2月6日終値）に日経ポジションを閉じる**などのリスク管理が言及されている。

### 3.2 テーマ別・セクター別

| テーマ | 内容・銘柄例（要各自調査） |
|--------|----------------------------|
| **防衛** | 17重点分野・骨太の方針で優先。地政学リスクは高止まり。三菱重工・IHI・川崎重工・日本製鉄等。PERは既に高いが「数十年の防衛費増」を前に折り込むと割高とは言い切れないとの見方。GCAP（日英伊戦闘機）・NATO離脱論で**非同盟国のリーダー**としての日本防衛産業が言及。 |
| **インフラ・建設・ゼネコン** | 高市政権に期待したいのは**インフラ**（上下水道・南関東トラフ・老朽化）。ドイツの80兆円規模のインフラ投資で株が急騰した例と対比。大林組・住友林業・JR東海等。**大阪都構想・リニア**（名古屋〜大阪）着工で大阪周辺・鉄道・空港（オリックス等）が物色されうる。 |
| **食品・小売** | 消費減税（食品ゼロ等）の恩恵。為替介入で円高寄りになれば食品株は買われやすい。**押し目や為替介入後の下げ局面**で買うセクターとして言及。 |
| **レアアース・レアメタル** | 南鳥島沖開発・中国依存脱却。**東京エンジニアリング**等が具体例。大型から資金が流れ、その後小型に波及するパターン。フィジカルAI・脱中国サプライチェーンの一環。 |
| **スポーツ** | 2026年はオリンピック・WBC・ワールドカップ等が重なり、**政治リスクが少なく下がりにくい**テーマとして初心者向けに言及。 |
| **埋もれていた株（バリュー）** | 造船・不動産・建設等、**インフレで売上が上がる**うえ、PBR・PERで割安な銘柄が**底上げ**されうる。日経平均だけ見ると高くても、**個別の割安株がうわーっと上がるステージ**に入りうる。 |

### 3.3 高一トレード・名目GDP

- 高市氏勝利が続く前提では、**高圧経済政策**（需給ギャップを高めに維持）が数年続くとの整理。**名目GDPと株価は連動**するため、**6〜8ヶ月は上昇**という基本線が複数ソースで語られている。
- **骨太の方針（6月閣議決定）**で**17重点分野の優先順位**が示されれば、**「すぐ実現する」「既に走っている」**分野に資金が集まりやすい。

### 3.4 国内株のまとめ

- **シナリオ**：与党過半数確保を前提に、2/3の有無で「一時的な押し（為替介入）」か「ストレート株高」を想定。過半数割れの場合は**暴落リスク**を前提にポジション・オプションでリスク管理。
- **テーマ**：防衛・インフラ・ゼネコン・食品（消費減税・円高時）・レアアース・大阪都構想・スポーツ・**割安バリュー**をバランスよく持つ。高市政権で既に動いた銘柄が、2/3取得で**推進力アップ**するとの見方あり。
- **リスク**：日経は**米国ハイテク（AI・半導体）の連れ高**を受けているため、2026年半ば以降の米国ハイテクの振れに**連動する変動**を想定する。

## 4. リスク管理・運用上の注意

- **選挙**：2月8日投票前は**日経の過大な建玉**を減らす、**オプションでリスク限定**するなどの選択肢が言及されている。
- **金利・債券**：長期金利上昇が続く想定では**長期債の目減り**が続く。債券は短め・または金・株で代替する考え方がある。
- **為替**：円安トレンドは構造的。**ドル建て資産（海外株・金ETF等）**は円ベースで価値が上がりやすい一方、**為替介入で一時円高**になれば円建て株（輸出株以外）は恩恵を受けうる。
- **分散**：**金・海外株・国内株**の3軸に加え、**地域（米国以外）・セクター（成長とバリュー・コモディティ）・時間（積立・押し目）**の分散を意識する。

## 6. 要約：3軸の配分イメージ（考え方）

- **金（貴金属）**：**ヘッジ・保険**として一定比率を維持。押し目積立・現物/ETFの併用。銀・銅等は産業需要テーマとしてサブで持つ選択肢。
- **海外株**：**分散**（S&Pとそれ以外のバランス）と**バリュー・中小型・素材・エネルギー**を意識。ハイテクは**銘柄選別**し、過剰投資リスクを踏まえて集中しすぎない。季節性（2月の不安定さ、6〜7月の日本優位）で再バランスのタイミングを検討。
- **国内株**：**選挙リスク**（過半数割れ時の暴落）をオプション・建玉削減で抑えつつ、**防衛・インフラ・食品・レアアース・大阪・割安バリュー**をテーマに分散。与党過半数〜2/3を前提に、一時押し（為替介入）後も**中長期は株高**とのシナリオを織り込む。

以上は複数ソースの**共通項・頻出テーマ**を基にした枠組みである。実際の配分・銘柄・売買は**自己責任**で、必要に応じて最新のレポート・データで更新すること。
