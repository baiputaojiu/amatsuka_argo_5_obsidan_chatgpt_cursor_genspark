## 2. CME手数料改定の衝撃

### 発表内容

| 項目 | 内容 |
|------|------|
| **発表日** | 2026年1月14日（水曜日） |
| **施行日** | 2026年2月2日 |
| **タイトル** | 金属バランスに関する手数料新設・改定（全ライフサイクルへの課金） |
| **対象品目** | 金、銀、プラチナ、パラジウム、銅、アルミニウム |
| **対象外** | マイクロ先物契約（現物受け渡しを伴わない） |

### 4つの手数料体系変更

| 手数料種別 | 従来 | 新料金 | 増加率 |
|------------|------|--------|--------|
| **①登録手数料**（倉庫入庫・バラント発行） | 数ドル〜無料 | **$100** | 大幅増 |
| **②管理手数料**（月額保管） | $15/月 | **$30/月** | 2倍 |
| **③譲渡手数料**（売買用） | $10〜25 | **$50** | 2〜5倍 |
| **④取り消し手数料**（現物引き出し） | $30〜50 | **$500** | **約16倍** |

### 取り消し手数料の意味

> [!danger] 事実上の「出銀制限」
> **バラントを解約して現物の銀を倉庫から引き出そうとすると$500の手数料**
> - 事実上の「出国税」
> - 現物持ち出しへのペナルティ
> - 今後さらに引き上げる可能性
> → **事実上の持ち出し禁止令**

### 誰がダメージを受けるか

> [!warning] アービトラージ業者への影響
> - 先物と現物のわずかな価格差（サヤ）を抜いて利益を得る業者
> - 市場の歪みを直す調整役
> - 利益は非常に薄い
> - $500の固定コストで利益が吹き飛ぶ → **赤字に**
> - 結果: 業者が市場から撤退
> - 調整役がいなくなると: **ペーパー先物価格と現物価格が乖離**

### 価格乖離の現実

> [!note] 二重価格の世界
> - 画面上のチャート価格: 92ドル（先物）
> - 実際のコインショップ: **120ドル以上でないと売ってくれない**

> [!important] CMEの狙い
> **「市場機能を壊してでも、アービトラージ業者を追い出してでも、倉庫から銀を出させたくない」**
> → それほどまでに裏側の銀不足は深刻

## 3. 世界的な銀在庫危機

### インド：在庫ゼロ

> [!danger] インド最大の貴金属精錬会社
> **「銀の在庫は完全に底をついた」**

> [!quote] キャリア27年のトレーディング責任者
> 「こんな狂った市場は見たことない。文字通り在庫がないんだ」

- 背景: ヒンドゥー教の祭典、SNSインフルエンサーの「銀を買え」煽り
- 国民が殺到している状況

### 中国：全てを吸引

- 中国の個人が銀に群がっている
- 爆買いが手伝っている
- コメント欄でも報告多数
- 世界中の銀が消費者・投資家に吸い上げられている

### ロンドン：銀行が逃亡

> [!danger] ブルームバーグ報道
> - 在庫不足があまりに深刻
> - 銀行員が顧客から「在庫をよこせ」と電話で怒鳴り散らされている
> - **大手銀行の中には価格変動と在庫不足を恐れて、値段を出せない・取引を拒否して逃げたところも**
> - **銀を借りるための金利が一時年200%を超えた**
> → 金融資産としてもはや機能していない

### CME倉庫の状況

| 項目 | 数値 |
|------|------|
| **CME総在庫** | 4.4億オンス |
| **2026年3月限の建玉** | **7.5億オンス** |
| **カバー率** | 在庫の**1.7倍**の契約が売りに出されている |

> [!warning] 異常事態
> - 3月限を1月限・2月限に切り上げる動きが出ている（異常）
> - みんなが「今すぐ在庫が欲しい」状態
> → CMEは手数料という名の「シャッター」を下ろした

## 4. 銀ETF選択：SLV vs PSLV

### 2021年シルバースクイーズ運動の教訓

> [!quote] 当時のネット掲示板
> **「SLVを売ってPSLVを買え」**

#### SLV（iShares銀ETF）の特徴

| 項目 | 内容 |
|------|------|
| **運営** | iShares（ブラックロック） |
| **カストディアン（保管者）** | **JPモルガン** |
| **現物裏付け** | あり |
| **流動性** | 高い |

> [!warning] 2021年の問題点
> - シルバースクイーズ運動はJPモルガンを破綻させることが目標
> - **SLVの銀保管者がJPモルガン自身**だった
> - 「敵にお金を預けても価格は上がらない」という主張
> - 2021年2月にSLVは目論見書をこっそり変更
> - **「銀の需要急増により十分な現物を調達できない可能性がある」**という警告文を追加

#### PSLV（Sprott銀ETF）の特徴

| 項目 | 内容 |
|------|------|
| **運営** | Sprott（スプロット） |
| **カストディアン** | カナダ造幣局 |
| **特徴** | 集まった資金で積極的に現物市場から銀を調達しロック |
| **透明性** | より高い |
| **現物引き出し** | 条件を満たせば可能（ルートが明確） |

> [!note] 創業者の立場
> スプロット氏は長年銀の価格操作を批判していたため、運動参加者から「同志」として見られた

### 実際のパフォーマンス比較

> [!tip] SLV vs PSLV
> **「ほとんど変わらない」**
> - 直近の価格推移を比較するとほぼ同じ動き
> - 違いが出るのは「究極的な局面」があった時
> - リスクをどう考えるか次第

### 日本の選択肢：1542（純銀上場信託）

| 項目 | 内容 |
|------|------|
| **運営** | 三菱UFJ信託銀行 |
| **シリーズ** | 金の果実・銀の果実 |
| **タイプ** | **国内現物保管型** |
| **安心感** | 高い |

> [!warning] 現物引き出しの条件
> - LBMA規格のラージバー1個分（約30kg）以上が必要
> - 現在の価格で**約1,462万円相当**
> - 転換手数料と配送料もかかる
> - 一般投資家には「無理ゲー」感あり

### 投資手段の選び方

> [!tip] と松氏の見解
> - **現物が最強**だが、保管・税金でめんどくさい面も
> - **スマートに手軽に**行きたいなら現物保管型ETFで十分
> - どれがいいかは「本人の考え方次第」
> - 情報を集めて自分に合ったやり方を見つけることが重要

## 6. ドンロー主義と通貨危機

### 全ての点を繋ぐ「一本の線」

> [!important] ドンロー主義（Donroe Doctrine）
> - **モンロー主義の現代版**
> - 「世界の警察官」からの完全撤退
> - **アメリカはアメリカ大陸とグリーンランド（地球の西半分）しか考えない**
> - 東アジアは見捨てられつつある

### 韓国ウォン危機

| 項目 | 内容 |
|------|------|
| **ウォン安** | 17年ぶりの安値 |
| **ベセント財務長官発言** | 「行き過ぎ」と異例の介入発言 |
| **キャピタルフライト** | 今年2週間で韓国個人投資家が**3,000億円以上**のウォンを売って米国株を購入 |
| **ドル国内還流なし** | 現代自動車・SKハイニックスが関税回避のため稼いだドルを米国工場建設に使用 |

> [!note] SKハイニックス・ハンファは絶好調
> - HBM（高帯域メモリ）のSKハイニックス
> - 防衛関連のハンファ
> - 一方サムスンはメモリ半導体開発で遅れ、微妙な立場

### 日本円危機

| 項目 | 内容 |
|------|------|
| **ドル円** | 159円45銭まで進行（160円目前） |
| **片山財務相発言** | 「極めて憂慮。あらゆる手段を排除しない」（レベル8） |
| **市場の反応** | 大勢は変わらず |
| **次の警戒ポイント** | 「レートチェック」という言葉（レベル9）→ レベル10が実際の介入 |

### 為替を見る覚え方

> [!tip] と松氏の解説
> **ドル円 = ドルと円のプロレス**
> - ドルが勝つ → チャート・数字が上がる
> - ドルが勝つ = 円が負ける = 円安
> - 今は160円方向に向かっている（ドルの勝ち）

### 「教科書が通用しない」悪い円安

| 教科書 | 現実 |
|--------|------|
| 日本が利上げすれば円高 | 日本は利上げした |
| 米国が利下げすれば円高 | 米国は利下げした |
| **結果は円高のはず** | **なぜか猛烈な円安が進行** |

### 円安の構造的要因

1. **高市政権の積極財政と量的緩和観測**（良いことだが円安要因）
2. **構造的な円の垂れ流し**: 貿易赤字・新NISA
3. **アジア全体の地盤沈下**: ドンロー主義による東アジア通貨の評価変化

> [!note] と松氏の見解
> **「日本にとっては厳しいが、ここが逆に勝負時。良い日本に変えていける。高市首相に期待がかかる」**
