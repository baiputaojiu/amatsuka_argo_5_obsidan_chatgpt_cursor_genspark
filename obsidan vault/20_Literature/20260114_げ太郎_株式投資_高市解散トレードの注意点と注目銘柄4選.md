## 1. 高市解散トレードの本質と注意点

### 1.1 高市解散の概要

- **2026年1月14日**: 高市首相が党幹部に衆院解散を伝達
- **2月上旬〜中旬**: 衆議院選挙の見込み
- 自民党の議席確保により、高市首相の政策がより通りやすくなる期待

### 1.2 なぜ「高市相場」で株が買われるのか

> [!tip] マーケットの期待
> - 金融緩和継続・金利引き下げ観測
> - 積極的な財政出動（お金のばらまき）
> - 核融合・サイバーセキュリティなど特定セクターへの国策支援

### 1.3 過去の高市トレードの教訓（2025年10月）

> [!warning] 重要な警告：天井は選挙前につく

| 日付 | イベント | 株価動向 |
|------|----------|----------|
| 2025年10月初旬 | 高市トレード開始 | 上昇 |
| 2025年10月4-5日 | 衆議院選挙 | - |
| **2025年10月21日** | 高市首相就任 | **天井** |
| 10月21日以降 | - | ずるずる下落 |

#### 高市銘柄の天井パターン

```
住友電気工業（5802）→ 10月21日に天井
FFRIセキュリティ（3692）→ 10月21日に天井
```

> [!important] げ太郎氏の見解
> **「10月21日（首相就任日）に天井をつけたことを皆覚えている。今回も選挙結果が出る前、もう少し前にはすでに天井をつけていて、選挙前後ぐらいでは横ばいかずるずる下げているのがメインシナリオ」**

### 1.4 実は「AI相場」が主因だった

> [!note] 重要な事実
> 2025年10月の「高市相場」と言われた上昇は、実際には**米国SOX指数（半導体指数）との相関が高い**

- 日経平均とSOX指数を重ねると、ほぼ同じ動き
- 高市効果は選挙前後の短期に限定
- 中長期の上昇はAI・半導体銘柄が主導

### 1.5 今回のトレード戦略

> [!danger] 絶対に守るべきポイント
> 1. **短期エントリーなら早めに売却**
> 2. **決算発表時期と選挙日程が重なる→決算前には絶対に振る（手仕舞い）**
> 3. 高値掴みは絶対に避ける

## 2. 年明け相場の状況

### 2.1 日経平均の動向

- **年明け初日から予想以上に強い**
- 5万4000円突破（史上最高値圏）
- 2025年のトランプ関税ショック時は3万1000円付近まで下落→驚異的な回復

### 2.2 上昇を牽引しているセクター

| セクター | 背景 |
|----------|------|
| **AI・半導体銘柄** | 引き続き強い |
| **建設系銘柄** | 年明けから特に高値更新銘柄多数 |
| **高市銘柄の一部** | 政策期待 |

### 2.3 建設業界の好調要因

> [!info] 建設業界の構造変化
> 1. **建設需要旺盛**
> 2. **人手不足** → 業者側の交渉力向上
> 3. **資材価格高騰** → 価格転嫁が進む
> 4. **利益率改善** → 「値段上げなければやりません」と言える立場に

## 3. 注目銘柄1：コプロホールディングス（7059）

### 銘柄概要

| 項目 | 内容 |
|------|------|
| **市場** | 東証プライム |
| **セクター** | サービス業（人材派遣） |
| **事業内容** | 建設業界向け人材派遣 |
| **配当利回り** | 約4% |
| **PER** | 約15〜16倍 |

### 投資テーマ

> [!tip] 建設業界の人手不足ど真ん中銘柄
> - 売上・利益ともに成長継続
> - 配当利回り高め
> - PERはそこまで高くない

### 足元の課題

> [!warning] 稼働率低下の理由
> - **技術者の採用は進んでいる**
> - **しかし営業人員が足りない**
> - → 案件獲得が追いつかず、派遣人員が余っている状況

### 投資戦略

- **短期**: 決算発表が近いため注意
- **中長期**: 営業と技術者のバランスが取れれば大きく業績改善
- **押し目買い**: 下げたタイミングでのエントリーが有効

### 投資テーマ：TOPIX組み入れイベント

> [!tip] TOPIX組み入れの仕組み
> 1. IPO銘柄は一定期間後にTOPIXに組み入れられる
> 2. 組み入れ日（1月末予定）に向けて**パッシブ資金の強制買い**が発生
> 3. TOPIX連動ETF・投信がSBI新生銀行を買わなければならない

### 投資戦略

| 項目 | 内容 |
|------|------|
| **エントリー推奨時期** | 1月上旬〜中旬 |
| **売却推奨時期** | TOPIX組み入れ日の**2〜3日前**（1週間前が目安） |
| **注意点** | 組み入れ日ギリギリまで持たない |

> [!warning] 過去の傾向
> 「TOPIX組み入れ銘柄は、組み入れ日の2〜3日前には天井をつけることが多い」

### 足元の動向

- 1月初旬に上昇 → 上がりすぎて一時押し目
- 今後1〜2週間で上昇圧力が再び出てくる可能性

## 5. 注目銘柄3：ノート（5243）

### 銘柄概要

| 項目 | 内容 |
|------|------|
| **市場** | 東証グロース |
| **セクター** | 情報・通信業 |
| **事業内容** | note（ブログ・記事投稿プラットフォーム）運営 |

### 直近の株価動向

- **2026年1月14日**: **+21.33%（ストップ高）**
- 前日の決算発表がポジティブサプライズ

### 投資テーマ

> [!tip] ノートの成長ドライバー
> 1. **生成AI効果**: 記事執筆のハードルが下がり、有料記事数が増加
> 2. **コスト一定のビジネスモデル**: 売上増→利益増がダイレクトに効く
> 3. **生成AIの引用元として高評価**: 日本語記事の引用元上位にnoteが位置
> 4. **季節性**: 1月は新年の勉強意欲で売上が上がりやすい（第1四半期が強い）

### ビジネスモデルの特徴

- **ストック型・プラットフォーム型**
- コストが一定なので、売上が上がれば利益率改善
- 今まで損益分岐点付近で推移していたが、黒字化が本格化

### 投資戦略

- 足元は急騰しており高値圏
- **押し目待ち**が有効
- 次の第1四半期決算も期待できる可能性

## 6. 注目銘柄4：ソルビー（7364）

### 銘柄概要

| 項目 | 内容 |
|------|------|
| **市場** | 東証スタンダード |
| **旧社名** | 日本リビング保障 |
| **セクター** | サービス業 |
| **事業内容** | 住宅設備の保守・保障業務 |
| **時価総額** | 約172億円 |

### ビジネスモデルの特徴

> [!info] ストックビジネスの安定性
> - 保証金を先にもらい、期間にわたって売上計上
> - **超安定したストックビジネス**
> - 一部セグメントにボラティリティあり

### 足元の状況

| 決算 | 内容 |
|------|------|
| 今期予想発表時 | 非常にポジティブ → 株価上昇 |
| 第1四半期決算 | 調子が悪い → **失望売り**で株価下落 |

### 会社の説明

> [!note] IR説明
> - 第1四半期の業績悪化は**一時的**
> - 過去案件の影響で利益率が悪化
> - **下期以降は大きく伸びる**と主張

### バリュエーション分析

| 指標 | 現在 | 2028年6月期目標 |
|------|------|-----------------|
| 時価総額 | 172億円 | - |
| 経常利益 | - | **50億円**（中期計画） |
| 純利益（推定） | - | 約35億円 |
| 想定時価総額（PER10倍） | - | 350億円（約2倍） |
| 想定時価総額（PER20倍） | - | 700億円（約4倍） |

> [!tip] ストックビジネスの再評価余地
> 保証系企業はこれまでPER10倍程度で放置されていたが、最近は「ストックビジネスだからPER20倍でも良いのでは」という再評価の動きも

### 投資戦略

- **第2四半期決算**: おそらくそこまで改善しない（会社も下期からと言っている）
- **部が悪くない勝負**: 第1四半期で失望売りが出ているため、よほど悪くない限り無風〜プラス
- **中長期**: 会社説明を信じるならアップサイド大きい
