# CME証拠金ルール変更と銀125ドルへの道｜3月Xデーに備えよ（2026年1月14日）

> [!summary] 概要
> 2026年1月14日の市場速報。CMEが2026年に入って連続して銀先物の証拠金を引き上げ、さらに算出方法を「パーセンテージ方式」に変更するという根底からのルール改定を実施。加えて現物受け渡しのない「100先物」を導入し、個人投資家を現物市場から切り離す「ホイホイ」を仕掛けている。CME在庫は急減し、3月21日前後の「Xデー」に向けてデフォルトリスクが高まる。ノーブルゴールドの銀125ドル予想と「一生に1度のスーパーサイクル」シナリオを解説。

## 2. CME証拠金ルール変更の詳細

### 2026年に入ってからの証拠金引き上げ履歴

| 日付 | 内容 | 増加率 |
|------|------|--------|
| 2025年12月27日頃 | 引き上げ開始 | - |
| 2026年1月1日 | 追加引き上げ | - |
| **2026年1月7日** | 銀証拠金を$32,500に引き上げ | **+47%** |
| **2026年1月9日** | さらなる引き上げ | **+30%** |
| **2026年1月13日** | 算出方法をパーセンテージ方式へ変更 | ルール根本改定 |

### 新パーセンテージ方式

| 商品 | 証拠金率（契約金額に対して） |
|------|------------------------------|
| **シルバー** | 9% |
| **プラチナ** | 9% |
| **ゴールド** | 5% |
| **パラジウム** | 11% |
| **投機目的（実需以外）** | +9.9〜10%上乗せ |

### パーセンテージ方式の意味

> [!danger] 自動ロスカットシステム
> - 従来: 証拠金は固定額、変更には通知が必要
> - 新方式: **銀価格が上がれば自動的に必要証拠金が引き上げられる**
> - 急騰した瞬間にレバレッジをかけている投資家を**強制ロスカットで焼き払う**ための自動システム

## 3. CMEの3つの罠

### 第1の罠：証拠金の連続引き上げ

- 2026年に入って10日半で複数回の引き上げ
- 個人投資家を振り落とす狙い

### 第2の罠：パーセンテージ方式への移行

- ルールの根底から変更
- 価格上昇時に自動で証拠金要求が増加
- 通知なしで投資家を追い出せる仕組み

### 第3の罠：100銀先物（ホイホイ）

> [!warning] 2026年1月13日発表
> - **開始日**: 2026年2月9日
> - **取引単位**: 100オンス（従来の1/50）
> - **現物引き渡し**: **不可能**（現金決済のみ）
> - **満期が来ても絶対に現物は渡さない**

#### ロビンフッド副社長のコメント

> [!quote] 表向きの説明
> 「我々の使命である金融民主化に沿って、少ない資本で銀市場に参加しやすくなる」

#### 実態

> [!danger] と松氏の分析
> - CMEはロビンフッド、プラス500US（CFD大手）と提携
> - **若者・初心者のマネーを現物市場に関係ないところへ誘導**
> - 「銀のふりをしたデジタル賭博場」への誘導
> - **現物流出を防ぐための罠＝ホイホイ**

## 4. 3月Xデーと銀125ドルシナリオ

### CME在庫の急減

```
2025年〜: 銀に注目が集まり在庫上昇
↓
現在: 急減（放物線を描くように指数関数的に減少）
↓
3月21日前後: Xデー到来か
```

### 数字で見る危機

| 項目 | 数値 |
|------|------|
| **CME総在庫** | 4.4億オンス |
| **2026年3月限の建玉** | **7.5億オンス** |
| **カバー率** | 在庫の**1.7倍**の契約 |
| **3月限の引き渡し期限** | **2026年3月21日前後** |

> [!danger] デフォルトリスク
> - 7.5億オンスの権利を持つ人たちの**半分でも現物を要求**すれば在庫は枯渇
> - CMEが100先物を作ったのは**3月末の破綻を回避するためのガス抜き**

### ノーブルゴールド・インベストメントの予想

#### 予測の信憑性

> [!note] 過去の実績
> 「短期的に価格が倍増する」と予測 → **実際に銀価格は2倍以上に高騰**

#### 次のターゲット：125ドル

> [!success] 一生に1度のスーパーサイクル

**著書「Silver Is The New Oil（銀は新しい石油である）」より**
- 銀は単なる貴金属から**エネルギー転換に不可欠な戦略的資源**へ変化

### 銀需要の4本柱

| 分野 | 内容 |
|------|------|
| **再生可能エネルギー** | 太陽光パネル |
| **EV** | バッテリー、パーツ |
| **AIハイテク** | 半導体チップ |
| **医療** | 抗菌作用を利用した医療機器（消費される） |

### 埋蔵量枯渇懸念：あと20年

| 論点 | 内容 |
|------|------|
| **累積赤字** | 過去4年で世界の年間消費は生産量を約7億オンス上回る |
| **供給の硬直性** | 銀の約70%は銅・亜鉛採掘の副産物 → 銀価格が上がっても増産できない |
| **消費型消失** | 金は宝飾品・延べ棒で保有されるが、銀は工業品として消費される |
| **リサイクル困難** | 単価が低いためコストが合わない → 市場から消えていく速度が金より早い |

## 5. 投資戦略と推奨事項

### CME/銀行の今後の戦術予想

| 戦術 | 内容 |
|------|------|
| **マージンの暴力** | 証拠金をさらに引き上げ、個人投資家を強制退場させる |
| **ホイホイ** | 100先物で小口投資家を現物から切り離す |
| **強制現金決済** | 「現物は渡せない。紙切れで我慢しろ」と宣言する可能性（不可抗力条項） |

### JPモルガン決算との関連

> [!note] と松氏の推測
> JPモルガンが巨額の引当金を積んだのは単なる不況への備えだけではないかもしれない。
> **コモディティ市場（特にシルバー）での巨額損失に備えている可能性**

### 推奨事項

#### 1. アンアロケート口座から脱出

> [!danger] 重要
> 銀行預かりの銀（アンアロケート）を持っているなら、すぐに：
> - **アロケート（現物特定）**にする
> - または**手元に引き出す**
> 
> 銀行が強制決済を宣言したら、アンアロケートの銀は戻ってこない

#### 2. CME先物には触らない

> [!warning] ルール変更リスク
> - ルールを勝手に変えられる場所での取引はギャンブル
> - 短期取引なら話は変わるが、基本は避ける

#### 3. 現物が最強

> [!tip] と松氏の結論
> **「銀行は持っていない。現物をあなたが持っていること、それが最大の武器であり資産防衛」**

#### 4. ETF選択の指針

| 選択肢 | 特徴 |
|--------|------|
| **国内在庫保管型ETF** | 三菱等が運営、一定の安心感あり |
| **投資信託** | 同上 |

> [!note] ETFの限界
> - 現物が調達できなくなったら打ち切りの可能性
> - 但し、それまでの上昇分の利益は得られる
> - 現物管理が大変な人にはスマートな選択肢
