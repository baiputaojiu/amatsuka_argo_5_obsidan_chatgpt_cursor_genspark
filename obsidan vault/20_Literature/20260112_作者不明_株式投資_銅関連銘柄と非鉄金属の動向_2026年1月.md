# 銅関連銘柄と非鉄金属の動向（2026年1月）

> [!summary] 概要
> 銅の国際価格が1万3000ドル台と歴史的高値に到達。AIデータセンター需要の急増と供給逼迫を背景に、資源メジャーの業界再編が加速。日本の非鉄金属銘柄も軒並み上場来高値を更新する力強い展開となっている。

---

## 銅価格の歴史的高騰

### 価格推移

| 時期 | 価格水準 | 備考 |
|------|---------|------|
| 2ヶ月前 | 1万ドル突破 | 「いよいよ面白くなる」と注目 |
| 2026年1月 | **1万3000ドル台** | 歴史的高値を記録 |

> [!tip] 上昇の背景
> - 堅実な需要増加（AI・データセンター）
> - 供給の逼迫
> - 長期的な構造変化

---

## 銅の特性と需要構造

### 銅が不可欠な理由

> [!info] 銅の優れた特性
> - **電導性**: 電気を効率よく通す
> - **熱伝導性**: 熱を効率よく伝える
> - **加工性**: 様々な形状に加工しやすい

### 主要な使用分野

```
送電線 ─────────┐
自動車 ─────────┤
家電製品 ───────┼─→ 現代社会のインフラを支える
データセンター ─┤
半導体の超微細配線 ┘
```

> [!important] 注目分野：AI・データセンター
> BHPグループのコメント：
> 「AI向けはまだ初期段階。データセンター投資拡大が今後銅需要をさらに押し上げるのは間違いなく、10〜15年後には無視できない大きさになる」

---

## 供給逼迫懸念と業界再編

### 2030年問題

> [!warning] 構造的な供給不足
> BHPグループは「2030年から35年にかけて構造的な供給不足に陥ることが現実味を帯びてきている」と警告。

### 資源メジャーの再編動向

| 年 | 動き | 詳細 |
|----|------|------|
| 2024年 | BHP → アングロ・アメリカン | 買収を図るも断念 |
| 2025年 | アングロ・アメリカン + テックリソーシズ | 経営統合発表、新会社「アングロテック」 |
| 2026年1月 | グレンコア + リオティント | 経営統合の協議中 |

> [!note] 再編の背景
> - 銅需要の高まり
> - 資源鉱山業界の需要変化
> - ポートフォリオ見直しの必要性
> - 国益も絡む大型M&A

## 日本の銅関連銘柄

### 銅価格との相関

> [!chart] 株価と銅価格の関係
> 多くの銅関連銘柄が銅価格の上昇以上にプレミアムが乗った形で推移。
> - AI・データセンター関連としての期待
> - レアアース関連としての期待
> → コモディティ単体以上のリターンが期待できる場面も

### 主要銘柄の分析

#### 1. 住友金属鉱山

| 項目 | 内容 |
|------|------|
| 特徴 | 金・ニッケルの非鉄金属大手、江戸時代の別子銅山が起源 |
| 株価（1/9終値） | **7,171円** |
| 動き | 11-12月の5,000円付近から1ヶ月強で7,000円台へ急騰 |
| チャート | カップ・ウィズ・ハンドル形成後、上値抵抗線をブレイク |
| 業績 | 今期：微減収・減益、来期：増収増益予想 |

> [!tip] テクニカル
> 週足でカップ・ウィズ・ハンドルを形成し、過去の高値である上値抵抗線を明確にブレイク。上場来高値を更新中。

#### 2. 三井金属

| 項目 | 内容 |
|------|------|
| 特徴 | 銅・亜鉛の精錬、各種電子材料。**AIデータセンター向け銅箔で世界有数のシェア** |
| 株価（1/9終値） | **19,900円** |
| 動き | 上場来高値を更新中 |
| 業績 | 今期：増収・大幅減益（前期の特別利益反動と特別損失計上）、来期：成長継続予想 |

> [!info] 成長計画
> AI通信インフラ用の特殊銅箔について、2030年度の利益を2025年度比約**2倍**にする計画を発表。

#### 3. 三菱マテリアル

| 項目 | 内容 |
|------|------|
| 特徴 | セメント、銅、電子材料、アルミ。チリ・インドネシア・カナダ・ペルーで銅鉱山出資 |
| 株価（1/9終値） | **3,997円** |
| チャート | 非常に力強い展開 |
| 業績 | 今期：減益（自動車向け不振、持分変動利益剥落、減損計上） |

#### 4. DOWAホールディングス

| 項目 | 内容 |
|------|------|
| 特徴 | 銅・亜鉛・レアメタル精錬、リサイクル、電子材料、金属加工、熱処理 |
| 株価（1/9終値） | **8,064円** |
| PER | **約11倍**（他銘柄の26倍に対して割安） |
| PBR | **約1.2倍** |
| 業績 | 太陽光向け電子材料は低迷、産廃処理は堅調、精錬は金属価格上昇と円安でカバー |

> [!note] バリュエーション
> 他の非鉄銘柄と比較して割安。ただし、AI・データセンター向けの材料が他銘柄ほど注目されていないことが要因か。

#### 5. 日鉄鉱業

| 項目 | 内容 |
|------|------|
| 特徴 | 日本製鉄系の資源会社、石灰石や銅の販売がメイン |
| 株価（1/9終値） | **2,624円** |
| チャート | 1990年バブル期の最高値を更新 |
| 業績 | 石灰石は価格改定、精錬収益縮小も銅価格上昇でカバー。新銅鉱山稼働予定 |

#### 6. 古河機械金属

| 項目 | 内容 |
|------|------|
| 特徴 | 足尾銅山が発祥、銅・金の金属部門＋土木建設機械・ロックドリル。レアアース回収機材開発中 |
| 株価 | 多少の上下をつけながら上昇トレンド |
| 業績 | 金属は堅調だが産業機械が不振、全体業績は厳しめ。来期は産業機械の回復見込み |

> [!note] レアアース関連
> 足元はレアアース関連として物色されている面もあり、業績よりモメンタム中心の動き。

## 投資の視点

### コモディティ投資 vs 関連株投資

> [!abstract] メリット・デメリット
> 
> **関連株投資のメリット**
> - コモディティ価格以上のプレミアム（期待値）が乗る可能性
> - AI・データセンター関連など複数のテーマで買われる
> 
> **関連株投資のデメリット**
> - 個別企業リスク（業績、経営判断）
> - コモディティ下落時に株価はより大きく下落する可能性
> 
> **対策**
> - ヘッジとして現物（コモディティETFや現物資産）も保有を検討
